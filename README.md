# Nuxt 3 with Net7 SPA Minimal Starter

## Setup

安裝 npm 相關 package

在專案根目錄，切換到 ClientApp 資料夾

執行:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

在 visual studio 執行 F5 偵錯模式

會以 spa 的方式執行 npm run dev

並且自動打開瀏覽器 proxy 到 `http://localhost:3000`

## Production

執行 .net 發佈
